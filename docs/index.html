mkdmg

The Fancy Apple Disk Image Creator

A powerful, modern CLI wrapper around hdiutil that makes creating, signing, and notarizing macOS Disk Images effortless.

Quick Start

# Create a DMG from a directory
mkdmg --volname "My App v1.0" "My App.dmg" ./build

# Use a JSON config for reproducible builds
mkdmg --config config.json

# Set a custom size and sign the image
mkdmg --disk-image-size 100 --codesign "Developer ID" out.dmg ./dist

# Dry run to preview commands
mkdmg --dry-run --verbose out.dmg ./dist
